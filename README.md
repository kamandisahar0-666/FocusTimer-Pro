# ⏳ FocusTimer Pro (PWA)

A professional, installable Pomodoro timer built with modern web technologies. This project focuses on high-quality UI/UX and seamless performance.

### 🚀 Key Features
- **PWA (Progressive Web App):** Installable on Desktop and Mobile.
- **Offline Support:** Powered by Service Workers for offline reliability.
- **Modern UI:** Glassmorphism design with optimized layering.

### 🛠 Technical Highlights
- **Advanced CSS:** Managed complex **Stacking Contexts** using `isolation: isolate` to ensure perfect UI element positioning.
- **Performance:** Optimized assets to meet **Google Lighthouse** PWA standards.
- **Responsiveness:** Fully responsive design for all screen sizes.

### 💻 Tech Stack
- HTML5, CSS3 (Modern Flexbox/Grid)
- Vanilla JavaScript
- Web Manifest & Service Workers
